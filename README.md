# Todo App 📝

A todo application built with vanilla JavaScript with full CRUD operations and data persistence.

## Features
- Add new tasks
- Delete tasks
- Mark tasks as completed (toggle)
- Data persists with localStorage
- Most recent tasks appear first

## Technologies
- HTML, CSS, JavaScript
- localStorage API
- DOM Manipulation

## Concepts Applied
- CRUD Operations
- localStorage (JSON.stringify / JSON.parse)
- Dynamic DOM Manipulation
- Event Listeners
- Single Responsibility Principle
